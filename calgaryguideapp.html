The Calgary Guide App

The internationally-respected medical reference is preparing to launch a mobile application.

Phase I

The early stages of this project aim to produce a working proof-of-concept mobile application to display cached, offline content from The Calgary Guide. For this early release, the application is only currently planned to be available on the Android platform.

Timeline: Spring 2018

Phase II

Analytics will be collected in a similar fashion to the current system used on the Calgary Guide website. Based on uptake, cross-platform development will start with the aim of building a platform-agnostic, more feature-rich version for release.

Timeline: Early 2019
