Anesthesia Logbook

Anesthesia Logbook is a web application developed to enable anesthesiogy residents in Canadian residency programs to track their clinical experiences in and out of the OR. The application provides

A web application which enables anesthesia learners to keep track of cases and procedures. Secure, efficient and mobile-friendly.

The project is in beta.
